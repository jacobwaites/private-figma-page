# private-figma-page
Simple express app to serve a figma url fullscreen behind a password.


# config
Set app `.env` 
```
COOKIE_SECRET=xxxx
SITE_PASSWORD=xxxx
SITE_TITLE=site title
FIGMA_URL=https://www.figma.com/xxxx
```
